## ⚡️ Quick Installation

### 🐋 Docker (Recommended)

```bash
# Clone the repository
git clone https://github.com/cosmicsec/cosmicsec.git
cd cosmicsec

# Start all 25+ microservices
docker-compose up -d

# Access the platform
# Frontend: http://localhost:3000
# API Gateway: http://localhost:8000
# API Docs: http://localhost:8000/docs
# GraphQL: http://localhost:8000/graphql
```

### 📦 Manual Installation

```bash
# Install Python dependencies
cd cosmicsec-core && pip install -r requirements.txt
cd ../cosmicsec-ai && pip install -r requirements/runtime.txt

# Install Node.js dependencies
cd ../cosmicsec-web/frontend && npm install

# Start services
python -m uvicorn cosmicsec_platform.main:app --port 8000  # API Gateway
python -m services.ai_service.main:app --port 8003        # AI Service
cd ../cosmicsec-web/frontend && npm run dev              # Frontend
```
