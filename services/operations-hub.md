# Operations Hub Service

**Phase 5 Operations Hub** — central operations management and orchestration service.

## Overview

The Operations Hub provides a unified interface for managing security operations across the platform.

## Features

- Centralized dashboard
- Orchestration workflows
- Integration with all microservices

## Endpoints

- `GET /health` — health check
- `GET /status` — service status

## Configuration

Environment variables:
- `PORT` — service port (default 8010)
