# Theme Builder Page#

**Theme Customization** — glassmorphism designer.

## Overview#

Customize CosmicSec's glassmorphism UI with real-time preview.

## Features#

- **Real-time preview** — see changes instantly#
- **Preset themes** — 4 built-in themes#
- **Custom colors** — primary, secondary, accent#
- **Blur intensity** — 10px-20px#
- **Gradient backgrounds** — linear gradients#
- **WCAG 2.1 AA** — accessibility compliant#

## Usage#

```typescript#
import { ThemeBuilderPage } from '@/pages';#

<ThemeBuilderPage#
  defaultTheme="cosmic-dark"#
/>#
```

## Available Themes#

- **Cosmic Dark** — default dark theme#
- **Midnight Blue** — blue-tinted dark#
- **Neon Nights** — vibrant neon accents#
- **Emerald Forest** — green-tinted dark#

## Customization Options#

- **Blur intensity** — backdrop blur radius#
- **Opacity** — glass panel opacity#
- **Border radius** — panel corners#
- **Gradient colors** — background gradient#
- **Typography** — font family/size#
